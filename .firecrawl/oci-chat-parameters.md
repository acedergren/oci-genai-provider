## Chat GenerativeAiInference

​`post /20231130/actions/chat`

Creates a response for the given conversation.

### Request

#### Chat

##### Parameters

| Name | Where | Description |
| --- | --- | --- |
| opc-retry-token | header | - Required: no<br>- Type: string<br>- Min Length: 1<br>- Max Length: 64<br>A token that uniquely identifies a request so it can be retried in case of a timeout or<br>server error without risk of executing that same action again. Retry tokens expire after 24<br>hours, but can be invalidated before that, in case of conflicting operations. For example, if a resource is deleted and purged from the system, then a retry of the original creation request<br>is rejected. |
| opc-request-id | header | - Required: no<br>- Type: string<br>The client request ID for tracing. |

##### Body

The request body must contain a single [ChatDetails](https://docs.oracle.com/en-us/iaas/api/#/EN/generative-ai-inference/20231130/datatypes/ChatDetails) resource.
