# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful environment for everyone. All participants in our community—contributors, maintainers, and users—deserve to engage without harassment or discrimination.

## Standards

### Expected Behavior

- **Be respectful** of differing viewpoints and experiences
- **Be constructive** in feedback and code reviews
- **Be welcoming** to newcomers
- **Be professional** in all interactions
- **Give credit** where it's due

### Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Sexualized language or imagery
- Trolling, insulting, or derogatory comments
- Publishing others' private information
- Violent threats or language
- Any conduct inappropriate for a professional setting

## Scope

This code applies to:
- Repository interactions (issues, pull requests, discussions)
- Community communications (email, chat)
- Project events
- Public representation of the project

## Reporting

If you experience or witness unacceptable behavior:

1. **Document** what happened (what, when, who)
2. **Report** privately to project maintainers
3. **Don't engage** further with the person if possible

All reports will be handled confidentially and investigated promptly.

## Enforcement

Depending on severity:

| Level | Action |
|-------|--------|
| Minor | Private warning |
| Moderate | Temporary ban from community spaces |
| Severe | Permanent removal from project |

## Examples

**Welcoming a newcomer:**
> "Welcome! Thanks for opening this issue. Here's some context that might help..."

**Constructive code review:**
> "This approach works. Have you considered X? It might improve performance."

**Respectful disagreement:**
> "I see your point, but I think approach X fits better because... What do you think?"
